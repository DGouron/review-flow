# Guide

Everything you need to get started with Reviewflow and configure it for your projects.

## Suggested Reading Order

If you're new to the project, follow this path:

1. **[Quick Start](./quick-start)** — Install, configure, and run your first automated review in minutes
2. **[Project Configuration](./project-config)** — Set up per-project review settings and skill assignments
3. **[Review Skills](./review-skills)** — Learn how audit agents work and how to create custom review skills
4. **[Troubleshooting](./troubleshooting)** — Common issues and how to resolve them

## Already Running?

If you have the system installed and need to fine-tune:

- Jump to [Project Configuration](./project-config) for per-project `.claude/reviews/config.json` setup
- Check [Review Skills](./review-skills) to customize or create new audit agents
- See the [Configuration Reference](/reference/config) for the full server config schema
